Features

  • Select recordings folder: allows to choose a directory where Recorder must save the .zip files it generates by recording.
  • Quit: closes the Recorder.
  • INF6903 project: opens the Web application at section #About.
  • Record: starts recording of business process activity. If a title is specified, it is saved in output files into generated .zip file then saved in Web application's database when recording is later uploaded. If a delay is specified, user gets X seconds to set up his environment before the recording starts. The recording starts at the instant the Recorder's window disappear, all monitors and user events are then recorded. The recording stops the instant the user presses Esc key on its keyboard, the Recorder's window reappears at this same instant and the recording output archive is saved in previously selected folder or default "recordings" folder at root of Recorder program's folder.
